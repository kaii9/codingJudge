# GoJudge Frontend

Next.js frontend scaffold for GoJudge.

## Requirements

- Node.js `^20.19.0 || ^22.13.0 || >=24.0.0`
- npm

The included `.nvmrc` selects Node.js `22.14.0`.

## Setup

```bash
nvm install
nvm use
npm ci
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Verification

```bash
npm run lint
npm run typecheck
npm run test:run
npm run build
```
